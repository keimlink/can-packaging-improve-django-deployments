Packaging

What is a Package?

A package is a distribution of software and data
in an archive file.

Package Metadata

  • Name
  • Version
  • Description
  • Author
  • Dependencies

Package Manager

Downloads and installs packages

Deployment

What is Deployment?

Make the Software
available to your Users

  1. Make a release
  2. Install the release

Make a Release

  • Bundle and minify JavaScript
  • Compile SCSS to CSS
  • Build the documentation
  • Compile translations
  • Remove files that should not be distributed

Then the release need to be stored somewhere.

Install the Release

  1. Transmit the release from the previous step
  2. Install the release and it's dependencies

Three steps
to prepare a Django project

              
myproject
├── manage.py # 2. Change myproject.settings to config.settings
├── myapp
│   ├── __init__.py
│   ├── admin.py
│   ├── apps.py
│   ├── migrations
│   │   └── __init__.py
│   ├── models.py
│   ├── tests.py
│   └── views.py
└── myproject # 1. Rename to config
    ├── __init__.py
    ├── settings.py
    ├── urls.py
    └── wsgi.py # 3. Change myproject.settings to config.settings

Add a setup.py file

              
from setuptools import setup
setup(
    name='myproject',
    version='1.0.0',
    description='Lorem ipsum dolor sit amet.',
    long_description='Consectetur adipisicing elit tempor.',
    author='Ada Lovelace',
    author_email='ada@example.com',
    include_package_data=True,
    install_requires=[],
    license='Other/Proprietary License',
    packages=['myproject'],
    zip_safe=False,
    classifiers=[
        'Framework :: Django',
        'Intended Audience :: Developers',
        'License :: Other/Proprietary License',
        'Operating System :: OS Independent',
        'Programming Language :: Python :: 3.4',
    ],
)

New project structure

              
myproject
├── manage.py           # 1. DJANGO_SETTINGS_MODULE modified to
|                       #    myproject.config.settings
├── myproject           # 2. New directory
│   ├── __init__.py     # 3. New empty file
│   ├── apps            # 4. New directory
│   │   ├── __init__.py # 5. New empty file
│   │   ├── myapp       # 6. myapp was moved into myproject/apps
│   │   │   ├── __init__.py
│   │   │   ├── apps.py
│   │   │   └── ...
│   └── config          # 7. config was moved into myproject
│       ├── __init__.py
│       ├── settings.py
│       │   # 8. INSTALLED_APPS modified to
│       │   #     myproject.apps.myapp.apps.MyappConfig
│       │   #    ROOT_URLCONF, WSGI_APPLICATION also modified
│       ├── urls.py     # 9. include() calls modified
│       └── wsgi.py     # 10. DJANGO_SETTINGS_MODULE modified
└── setup.py
